Форма авторизации uLogin через социальные сети. Улучшенный аналог loginza.

== Description ==

uLogin — это инструмент, который позволяет пользователям получить единый доступ к различным Интернет-сервисам без необходимости повторной регистрации,
а владельцам сайтов — получить дополнительный приток клиентов из социальных сетей и популярных порталов (Google, Яндекс, Mail.ru, ВКонтакте, Facebook и др.)

== Installation ==

1. Распакуйте содержимое папки "ulogin_install" в корень папки с платформой Magento.
2. Очистите кеш системы (System -> Cache Management -> Flush Magento Cache) и перелогиньтесь в роли администратора.
3. Создайте свой виджет на сайте http://ulogin.ru/ (описание см. ниже)
4. В админке: system > configuration > ulogin в полях "ID виджета для входа" и "ID виджета синхронизации аккаунтов" укажите соответствующие значения параметра "uLogin ID" Вашего виджета.
5. Сохраните изменения.

Вы можете создать свой виджет и редактировать его самостоятельно на сайте http://ulogin.ru/. 
Для создания виджета необходимо: 

1. Войти в Личный Кабинет (ЛК) на сайте http://ulogin.ru/
2. Добавить свой сайт к списку "Мои сайты"
3. На вкладке "Виджеты" добавить новый виджет
4. Внимание: Для работы модуля uLogin на платформе Magento в ЛК необходимо включить в обязательных полях профиля поле Еmail
5. В настройках модуля uLogin необходимо указать ID Вашего виджета в соответствующих полях
